ar1_mm <- function(x) {
  mu <- mean(x)
  sigma2 <- var(x)
  rho <- acf(x, 1, plot = FALSE)$acf[2]
  phi <- rho
  sigma2_eps <- sigma2*(1 - phi^2)

  colnames(parametros) <- c("mu", "sigma2", "sigma2_eps", "phi")
  return(parametros)
}

Teorema

MÃ¡xima verossimilhanÃ§a (MV)

O mÃ©todo de MV requer que saibamos a distribuiÃ§Ã£o de \(\epsilon_t\). Assumindo normalidade1, temos:

\[Y_t = \mu + \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + \cdots + \phi_p Y_{t-p} + \epsilon_t, \quad \epsilon_t \sim N(0, \sigma^2_{\epsilon})\]

Como \(\epsilon_1, \cdots, \epsilon_n\) sÃ£o iid, sua distribuiÃ§Ã£o conjunta Ã© dada por:

\[f(\epsilon_1, \epsilon_2, \cdots, \epsilon_n| \Theta) = \displaystyle \prod_{i = 1}^n \dfrac{\exp(-\epsilon_i^2/2\sigma^2_{\epsilon})}{\sqrt{2 \pi \sigma^2_{\epsilon}}}= \dfrac{\exp \Big(- \dfrac{1}{2\sigma^2_{\epsilon}} \displaystyle \sum_{i = 1}^n \epsilon_i^2 \Big)}{(2 \pi \sigma^2_{\epsilon})^{n/2}},\] em que \(\Theta = (\mu, \sigma^2_{\epsilon}, \phi_1, \cdots, \phi_p)\)

  1. O mÃ©todo nÃ£o se restringe apenas Ã  suposiÃ§Ã£o de normalidade e outras distribuiÃ§Ãµes podem tambÃ©m ser utilizadas.

MÃ¡xima verossimilhanÃ§a (MV)

Note que \(\epsilon_i = Y_i - \mu - \phi_1 Y_{i-1} - \cdots - \phi_p Y_{i-p}\), entÃ£o

\[f(\epsilon_1, \epsilon_2, \cdots, \epsilon_n| \Theta) = \dfrac{\exp \Big(- \dfrac{1}{2\sigma^2_{\epsilon}} \displaystyle \sum_{i = 1}^n (Y_i - \mu - \phi_1 Y_{i-1} - \cdots - \phi_p Y_{i-p})^2 \Big)}{(2 \pi \sigma^2_{\epsilon})^{n/2}},\]

MÃ¡xima verossimilhanÃ§a (MV) condicional

Para poder estimar os parÃ¢metros por mÃ¡xima versomilhanÃ§a, precisamos conhecer \(y_1, \cdots, y_n\), mas tambÃ©m (assumir que conhecemos) \(y_0, y_{-1}, \cdots, y_{-p}\). Assim, a verosimilhanÃ§a Ã© dada por:

\[L(\mu, \sigma^2_{\epsilon}, \phi_1, \cdots, \phi_p | y_{-p}, \cdots, y_n) = \dfrac{\exp \Big(- \dfrac{1}{2\sigma^2_{\epsilon}} \displaystyle \sum_{i = 1}^n (y_i - \mu - \phi_1 y_{i-1} - \cdots - \phi_p y_{i-p})^2 \Big)}{(2 \pi \sigma^2_{\epsilon})^{n/2}}\]

E a log-verosimilhanÃ§a Ã© dada por

\[l(\mu, \sigma^2_{\epsilon}, \phi_1, \cdots, \phi_p | y_{-p}, \cdots, y_n) = - \dfrac{n}{2} \log(2 \pi \sigma^2_{\epsilon}) - \dfrac{1}{2\sigma^2_{\epsilon}} \displaystyle \sum_{i = 1}^n (y_i - \mu - \phi_1 y_{i-1} - \cdots - \phi_p y_{i-p})^2.\]

MÃ¡xima verossimilhanÃ§a (MV) condicional

Assim, os estimadores de mÃ¡xima verossimilhanÃ§a (EMV), \(\hat{\Theta} = (\hat{\mu}, \hat{\sigma}^2_{\epsilon}, \hat{\phi}_1, \cdots, \hat{\phi}_p)\), sÃ£o aqueles que maximizam \[l(\Theta | y_{-p}, \cdots, y_n),\] ou , equivalentemente, os valores que minimizam \[-l(\Theta | y_{-p}, \cdots, y_n).\]

\[-l(\Theta| y_{-p}, \cdots, y_n) = \dfrac{n}{2} \log(2 \pi \sigma^2_{\epsilon}) + \dfrac{1}{2\sigma^2_{\epsilon}} \displaystyle \underbrace{\sum_{i = 1}^n (y_i - \mu - \phi_1 y_{i-1} - \cdots - \phi_p y_{i-p})^2}_{S(\mu, \phi_1, \cdots, \phi_p |y)}.\]

Para um \(\sigma^2_{\epsilon}\) qualquer, \(\hat{\mu}\) e \(\hat{\phi}s\) sÃ£o obtidos minimizando \(S(\mu, \phi_1, \cdots, \phi_p |y_{-p}, \cdots, y_n).\)

MÃ¡xima verossimilhanÃ§a (MV) condicional

Uma vez obtidos \(\hat{\mu}\) e \(\hat{\phi}s\), minimizamos \(-l(\sigma^2_{\epsilon}| \hat{\mu}, \hat{\phi}s, y)\). EntÃ£o, \[\hat{\sigma}^2_{\epsilon} = \dfrac{S(\hat{\mu}, \hat{\phi}_1, \cdots, \hat{\phi_p}|y_{-p}, \cdots, y_n)}{n}.\]

  • Na prÃ¡tica, para podermos implementar o mÃ©todo, precisamos os valores \(y_0, y_{-1}, \cdots, y_{-p}\) (valores que nÃ£o temos!).
  • Uma soluÃ§Ã£o Ã© fazer \(y_{0} = \cdots = y_{-p} = \bar{y}\)
  • Outra alternativa Ã© calcular a verosimilhanÃ§a a partir de \(i \geq p + 1\).
  • Este mÃ©todo, que condiciona nos valores iniciais, Ã© chamado de mÃ¡xima verossimilhanÃ§a condicional.
  • Quando \(\epsilon_t \sim N(0, \sigma^2_{\epsilon})\), os EMV para \(\mu, \phi_1, \cdots, \phi_p\) coincidem com os estimadores obtidos pelo mÃ©todo de mÃ­nimo quadrados condicional.

mv_ar <- function(x, p) {
  mu <- mean(x)
  n <- length(x)
  X <- matrix(mu, ncol = p + 1, nrow = n)
  Y <- matrix(x, ncol = 1)
  X[, 1] <- 1
  for (i in 2:(p + 1)) {
    X[i:n, i] <- x[1:(n - i + 1)]
  }
  betas <- solve(t(X) %*% X) %*% t(X) %*% Y
  sigma2_e <- sum((Y - X %*% betas)^2) / n
  parametros <- c(betas, sigma2_e)
  return(parametros)
}
